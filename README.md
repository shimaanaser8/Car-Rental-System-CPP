# Car Rental System in C++

A simple Car Rental System implemented in C++ that demonstrates the use of basic Data Structures, including Doubly Linked List and Stack.

## Features
- Insert a new car at the beginning, end, or middle
- Delete a car from beginning, end, or specific position
- Search cars by brand, model, price, or color
- Update car details
- Sort cars by day price
- Stack operations (Push, Pop, Peak, IsEmpty, IsFull)
